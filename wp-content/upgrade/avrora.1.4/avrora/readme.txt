== Installation ==

1. Upload the `Avrora` folder to the `/wp-content/themes/` directory
Activation and Use
1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme options
3. Please follow next steps for adding contact page:
    a.Go to dashboard, then create a page. Give it a name "Contact" or "Contact us" as you wish.
    b.Now on the right hand side you can see <strong>Template</strong> Drop-down. Select Contact Page Template
    c.Copy the contact form 7 shortcode and paste it in this page.
    d.Click on >Publish
